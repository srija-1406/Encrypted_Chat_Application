Encrypted Chat Application

This project was developed during my internship at Syntecxhub. It implements a secure chat system using encryption and socket programming concepts.

Features
- Uses AES encryption (Fernet) for secure communication
- Supports multiple clients using threading
- Implements TCP socket programming
- Ensures messages are transmitted securely

Technologies Used
- Python
- Cryptography library
- Socket programming
- Threading

How to Run
1. Run the program
2. Select option 1 to generate the key
3. Select option 2 to start the server
4. Select option 3 to start one or more clients
